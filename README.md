# AGU - Personal Management System (AGU-PMS)

Hệ thống hỗ trợ lập kế hoạch công tác dành riêng cho Giảng viên Trường Đại học An Giang.

## 🚀 Tính năng chính
- **Quản lý nhiệm vụ (Kanban):** Kéo thả công việc giữa các trạng thái Chưa làm - Đang làm - Hoàn thành.
- **Phân loại nhiệm vụ:** Tách biệt giữa Giảng dạy, Nghiên cứu (NCKH) và Hành chính.
- **Cơ sở dữ liệu sẵn sàng:** Cấu trúc SQL đã được tối ưu hóa cho giảng viên.

## 🛠 Công nghệ sử dụng
- **Frontend:** HTML5, CSS3, JavaScript.
- **Icons:** FontAwesome 6.0.
- **Backend (Gợi ý):** SQL Server.

## 📁 Cấu trúc thư mục
- `index.html`: Giao diện cấu trúc chính.
- `main.css`: Định dạng phong cách thương hiệu.
- `script.js`: Xử lý tương tác người dùng.

## 📝 Hướng dẫn cài đặt
1. Tải về hoặc Clone thư mục dự án.
2. Mở file `index.html` trên bất kỳ trình duyệt nào.
